# Fretboard — контекст проекта для Claude

## Что это
Интерактивный визуализатор гитарного грифа. Single-page web app без фреймворков.
Пользователь выбирает тонику и лад — ноты подсвечиваются на грифе. Можно кликать по нотам — они звучат.

## Файлы проекта
```
index.html          # Основной файл (рус.) — HTML + CSS + JS + base64 аудио (~467 KB)
index-en.html       # Английская версия (~461 KB)
chords.js           # База аккордов (CHORD_DB), аппликатуры относительно C
triads.js           # Триады: интервалы, инверсии, струнные сеты
arpeggios.js        # Арпеджио: типы и интервалы
samples/            # Исходные WAV файлы (не используются в рантайме — вшиты в HTML)
```

## Технологии
- Чистый HTML/CSS/JS, без фреймворков, без сборщиков
- Аудио: 2 base64-WAV сэмпла из Ableton, pitch-shift через Web Audio API `playbackRate`
- Работает локально с `file://` без сервера

## Режимы приложения

### Простой режим
- Гриф 12 ладов, стандартный строй (E A D G B E)
- 12 кнопок тоники + 7 кнопок лада (Ionian, Dorian, Phrygian, Lydian, Mixolydian, Aeolian, Locrian)
- Клик по ноте — звук

### Расширенный режим
- Гриф 24 лада
- Show All Notes — недиатонические ноты как fade
- Show Intervals — показывает интервалы вместо нот (R, b2, 2, b3...)
- Цвет нот по качеству: синий (мажорные лады), фиолетовый (минорные)
- Круг квинт — интерактивный SVG
- Блок Chords / Triads / Arpeggios

## Дизайн
Ультра-минимализм: фон `#262624`, плоские цвета, моноширинный шрифт, без градиентов/теней/скруглений.
